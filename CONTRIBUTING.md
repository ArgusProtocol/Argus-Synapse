# Contributing to Argus Protocol

We welcome contributions from Protocol Engineers, Distributed Systems Researchers, and Machine Learning Experts.

## Code of Conduct
Please be respectful and professional in all communications.

## Development Setup
Follow the instructions in the [README.md](./README.md) to set up your local environment.

## Pull Request Process
1.  Fork the repository.
2.  Create a feature branch.
3.  Ensure all Rust and Python tests pass.
4.  Submit a Pull Request with a detailed description of the changes.

## Areas for Contribution
-   **Task 5**: Multi-agent consensus for distributed gateways.
-   **Task 6**: Hardware-accelerated linearization via CUDA.
-   **Task 7**: Zero-Knowledge proof generation for $k$-coloring validity.
